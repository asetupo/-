# Программа бронирования туров

Это веб-приложение для просмотра каталога туров и бронирования выбранных туров.

---

## Описание

Данное приложение позволяет пользователям:
- Просматривать список доступных туров
- Выбирать дату для бронирования
- Оформлять бронирование

---

## Технологии

- Python 3.14
- Flask (микрофреймворк для веб-разработки)
- HTML, Jinja2 (шаблонизатор Flask)
- SQLite

---

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы программы

```bash
git clone <ваш репозиторий>
cd <папка проекта>
```

2. Создайте виртуальное окружение и активируйте его

```bash
python -m venv venv
source venv/bin/activate  # для Linux/macOS
venv\Scripts\activate     # для Windows
```

3. Установите необходимые зависимости

```bash
pip install -r requirements.txt
```

4. Запустите приложение

```bash
flask run
```

или

```bash
python app.py
```

5. Перейдите в браузере по адресу: [http://127.0.0.1:5000](http://127.0.0.1:5000)

---


## Дополнительная информация

- Убедитесь, что база данных настроена корректно.
- Вы можете расширять функционал, добавляя новые туры, пользователей и т.п.
